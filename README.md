# Wardcom Comptabilité-Accounting

Professional accounting and taxation services website for Wardcom Accounting Inc., serving Montreal and the greater Canadian community since 1998.

## Features

- Multilingual support (English, French, Arabic)
- Responsive design with Tailwind CSS
- Modern React 19 with TypeScript
- Vite for fast development and building

## Services Offered

- **Accounting**: Financial statements, compilation engagements, complete bookkeeping (payroll, GST-QST)
- **Taxation**: Consulting, income tax returns for companies, self-employed individuals, and personal taxes

## Run Locally

**Prerequisites:** Node.js (v18 or higher)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
